Caller Verification – Decision Guide

Interactive helper for Voice Match and Standard (Voice Mismatch).

1) Choose Method

2) Base Checks

Per policy: a miss for the security code (if available) is an automatic failed verification.
General rule: only one miss of account number, last 4 of SSN, or OTP may be replaced by an Additional Verifier. Two misses among those = failed verification.

3) Scenarios (Voice Match)

If any scenario below applies, it requires One-Time Passcode or one Additional Verifier (per your Voice Match guide).

3) Scenarios (Standard Verification)

For any scenario below, obtain one (1) Additional Verifier.

4) Additional Verification Options

Choose from the policy-approved list. For most cases you will need one (1) in total. If using Voice Match scenarios, either OTP or one Additional Verifier is acceptable.

CEC Verification Policy

Voice Biometric Verification – Voice Match

  1. First and Last Name
  2. If IVR Unverified, obtain account number AND last four digits of SSN.
  3. Security Code, if applicable
  4. The following scenarios require One-Time Passcode or one Additional Verifier:
    • Check Order – Address Change within 30 Day
    • Debit Card – Expedited Request
    • Debit Card – Address Change within 30 Days
    • Maintenance – Phone Number Change/Removal
    • Online Banking – Remove Lockout/Unlock
    • Zelle – Maintenance

Standard Verification

  1. First and last name.
  2. If IVR Unverified, obtain account number AND last four digits of SSN.
  3. Security Code, if applicable.
  4. One-Time Passcode (OTP).
    1. If OTP is Missed or Not Obtained:
      1. Click the Manual Button and select Date of Birth or mother’s maiden name
  5. Obtain one Additional Verifier for any of the following scenarios:
    1. Fraud alert/ID theft and no security code.
    2. Any one miss of either account number, last 4 of SSN, or OTP
    3. Only one miss is allowed to be replaced by an Additional Verifier per interaction.

After proper verification you may offer to add a security code as an added protection for the client.

Additional Verification

When using standard verification where OTP is not successfully verified, and in certain scenarios, additional verification is required. Select the required number of additional verifiers from the additional verification options below.

Additional Verifiers Required: One (1)

Scenarios:
  • Check Order – Address Change within 30 Days
  • Debit Card – Expedited Request
  • Debit Card – Limit Increase
  • Debit Card – Access Code Count Reset
  • Debit Card – Address Change within 30 Days
  • HELOC – Inquiries
  • IVR – Access Code Reset
  • Maintenance – Mailing Address Change
  • Maintenance – Email Address Change
  • Maintenance – Phone Number Change/Removal
  • Maintenance – Account Status Change
  • OLB – Establish/Reset Challenge Questions
  • OLB – Remove Lockout/Unlock
  • Tokenization – Mobile Wallet
  • Transfers – Internal Account
  • Funds Transfers
  • Travel Notes – Add/Remove
  • Zelle – Maintenance
Additional Verification Options:
  • Driver’s license, state identification, or passport number (expiration date not required).
  • Amount of last deposit (within $100).
  • Review of recent transaction history since last statement.
  • Company name and date associated with a recurring debit or credit.
  • Company name and amount (within $100) associated with a recurring debit or credit.
  • Last four digits of the ATM/Debit card number (except with Travel Notes/Debit Card scenarios).
This tool does not store any data. Use professional judgment and follow supervisor guidance for exceptions.
